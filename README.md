## Setup

```sh
$ git clone https://github.com/2gbeh/realestay.git
$ cd realestay
```

#### Client Setup

```sh
$ cd realestay/web
$ npm cache clean --force
$ npm install
# OR npm install --legacy-peer-deps

TODO: Update .env credentials

$ npm run dev
```

> Client: http://localhost:3000

#### Server Setup

```sh

$ npm cache clean --force
```
