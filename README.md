## Setup

```sh
$ git clone https://github.com/2gbeh/pinksl.ip.git
$ cd pinksl.ip
```

#### Client Setup

```sh
$ cd web
$ npm cache clean --force
$ npm install
# OR npm install --legacy-peer-deps

TODO: Update .env credentials

$ npm run dev
```

> Client: http://localhost:3000

#### Server Setup

```sh
$ cd api
$ npm cache clean --force
$ npm install
# OR npm install --legacy-peer-deps

TODO: Update .env credentials

$ npm run dev
```

## Usage

> Server: http://localhost:8000/api/graphql
